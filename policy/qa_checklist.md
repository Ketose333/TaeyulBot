# QA Checklist (범용)

## 공통 회귀
- [ ] 핵심 플로우 시작 → 종료가 끊기지 않는다.
- [ ] 새로고침/재진입 시 세션 복구가 의도대로 동작한다.
- [ ] 권한 없는 액션은 서버에서 차단된다.
- [ ] 경계값 입력(빈값/손상값/최소·최대 위반)에서 앱이 죽지 않고 에러만 노출된다.
- [ ] JSON/비JSON 실패 응답 모두 사용자 문구가 일관되다.

## UI/UX 회귀
- [ ] 오버레이 의미가 유지된다 (blocking/modal/drawer).
- [ ] ESC, 포커스, 스크롤 잠금, z-index 충돌이 없다.
- [ ] 버튼/입력/텍스트 스타일 일관성이 유지된다.

## 상태 저장소 회귀
- [ ] 외부 저장소(on) 동작 정상.
- [ ] fallback(off) 동작 정상.
- [ ] 재시작 후 fallback 휘발 여부가 의도와 일치한다.

## TCG 확장 스모크
- [ ] 덱 코드(v2) 정상/오류 케이스 처리 일관.
- [ ] deck-hub 게시/조회/검색/정렬 정상.
- [ ] 인증/로그아웃 분기 및 접근 제어 정상.
