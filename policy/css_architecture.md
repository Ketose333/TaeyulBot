# CSS Architecture (범용 책임표)

## 원칙
- 공용은 전역 스타일 파일에만 둔다.
- 페이지 전용은 페이지 파일에서만 관리한다.
- 색상은 토큰 단일소스에서만 정의한다.
- 동일 셀렉터/유틸 중복 정의 금지.

## 권장 계층
- palette: 색상/의미 토큰(`--p-*`, `--c-*`)
- styles: 타이포/입력/버튼/공용 유틸
- page css: 화면별 레이아웃/컴포넌트

## 금지
- 페이지 CSS hex 하드코딩
- 전역 파일에 페이지 특정 셀렉터 추가
- 공용 유틸을 페이지 CSS에서 재정의

## 체크
1. 공용/페이지 분류 먼저
2. 색상은 토큰 사용
3. 중복 정의 탐지
4. 빌드 및 화면 회귀 확인
