# policy/media.md — 미디어/파일 정책 단일본

## 저장/첨부 경로
- 미디어 원본 저장소는 `/home/user/.openclaw/media/` 하위를 사용한다.
- 디스코드 첨부는 `/home/user/.openclaw/media/` 하위 파일만 사용한다.

## 파일명 규칙
- 시간값 없이 주제/변형 중심 슬러그를 사용한다: `<topic>_<variant>.<ext>`
- 동일 이름 충돌 시 `_2`, `_3`처럼 숫자 suffix를 증가시킨다.

## 첨부/업로드 원칙
- 요청 주제와 일치하는 파일만 전송한다.
- 다중 후보일 때는 사용자 선택 후 전송한다.
- 채널 구분 없이 기본은 파일 단독 업로드(`NO_REPLY`)를 사용한다.
- 사용자가 설명을 요청한 경우에만 짧은 한 줄을 보충한다.
