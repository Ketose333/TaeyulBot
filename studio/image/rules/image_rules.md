# image_rules.md

한태율 이미지 생성의 단일 규칙 소스다.
프롬프트 가드는 이 문서 섹션만 사용한다.
상위 운영 정책은 [`policy/studio.md`](../../../policy/studio.md)를 따른다.

## COMMON_IDENTITY_LOCK
- reference image와 동일 인물/캐릭터 정체성을 유지한다.
- 얼굴 비율·헤어라인·핵심 인상은 고정하고, 포즈/표정/의상/장면 요소만 변형한다.
- 정체성은 유지하되 표정/무드/디테일은 요청에 맞게 유연 반영한다.

## REF_IMAGE_POLICY
- 생성 결과물 경로(`~/.openclaw/media/image/*`)는 ref-image로 사용하지 않는다.
- 아바타/원본 기준 이미지만 ref로 사용한다.

## REAL_STYLE_GUARD
- 실사(photorealistic) 톤을 유지하고 2D/카툰/셀셰이딩은 금지한다.
- 자연스럽고 담백한 광원/렌더 톤을 기본 기준으로 강제한다.
- 어두운 밤 배경에서는 로우키(low-key) 조명 톤(배경은 눌리고 인물은 부드럽게 분리)을 우선 적용한다.
- 과한 색보정/필터/글리치/렌즈왜곡 금지.
- 과한 대비/광택/시네마틱 톤으로 AI 티가 강하면 실패로 간주하고 재생성한다.

## TWO_D_STYLE_GUARD
- 2D/chibi 스타일을 허용한다.
- 2D 정체성은 디스코드 봇 프사 기준으로 일치시킨다.

## FRAMING_AND_POSE_BASELINE
- 기본 프레이밍은 흉상~상반신 클로즈업으로 유지한다.
- 인물 화면 점유율은 60~75%를 기본으로 유지한다.
- 얼굴 각도/시선은 요청 장면에 맞춰 자연스럽게 유지한다.

## BACKGROUND_QUALITY_BASELINE
- 읽히는 텍스트/로고/브랜드 표시는 기본적으로 금지한다.
- 장면은 과밀 소품/과장 장식/복잡 패턴 누적을 피한다.
- 실내/야외 모두 피사체 집중을 해치지 않게 배경 복잡도를 낮춘다.
- 실내 장면에서 우드톤(원목 벽/마루/우드 패널/우드 가구 메인 톤)은 기본적으로 금지한다.
- 우드톤이 메인으로 검출되면 실패로 간주하고 재생성한다.
- 의상 표면의 읽히는 문자/로고/브랜드 프린트도 기본적으로 금지한다(무지 패브릭 우선).
- 의상에 텍스트/로고 흔적이 보이면 실패로 간주하고 재생성한다.
- 실내 장면은 완전 무지 배경 대신 실제 거주공간의 약한 생활 흔적을 최대 0~1개만 허용한다.
- 모형/미니어처/세트장처럼 보이는 인공적 배경 질감이 강하면 실패로 간주하고 재생성한다.
- 장식 요소가 과다하거나 배경 디테일이 뭉개져 인물 분리가 약하면 실패로 간주하고 재생성한다.

## NANO_BANANA_PRO_GUARD
- `nano-banana-pro-preview` 사용 시 reference 일치도를 최우선으로 유지한다.
- 요청 없는 성별/인종/연령/얼굴구조 변화 금지.

## HARD_CASE_AVOIDANCE
- 고난도 패턴 의상, 반사·투명·발광 소재, 얼굴 가림 포즈가 메인으로 검출되면 실패로 간주하고 재생성한다.
- 사용자 장소 명시가 없을 때 실내 무드를 강제하지 않는다. 장식이 적은 중립 배경(실내/야외 모두 가능)을 기본으로 허용한다.
- 사용자가 야외/특정 장소를 명시하면 해당 장소를 우선 적용(실내 기본값보다 우선).
- 야외에서 군중·네온 간판 다량·복잡 텍스트 배경이 메인이면 실패로 간주하고 재생성한다.
- 사용자가 '인적 없는 곳/으슥한 곳'을 명시하면 간판·브랜드·상업 텍스트가 드러나는 배경은 우선 회피한다.
- 야간 장면에서 조명 기준이 모호하면 로우키 톤을 먼저 적용하고, 인물 가독성이 낮을 때만 완화한다.
- 사용자 전신 명시가 없는데 전신 위주로 생성되면 실패로 간주하고 재생성한다.

## OUTFIT_ONLY_LOCK
- 의상 변경 요청에서는 얼굴/헤어/비율/정체성 변경을 절대 금지한다.
- 변경 범위는 의상/소품/텍스처/색상으로 제한한다.
- 데님 메인 코디는 사용자 명시 요청이 없으면 실패로 간주하고 재생성한다.
- 저지/슬럽 코튼류 티셔츠 질감은 사용자 명시 요청이 없으면 실패로 간주하고 재생성한다.
- 나일론 바람막이/폴리 트윌 계열의 인공 광택 질감은 사용자 명시 요청이 없으면 실패로 간주하고 재생성한다.
